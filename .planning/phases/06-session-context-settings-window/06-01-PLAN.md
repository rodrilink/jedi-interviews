## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| settings renderer → main (IPC) | Untrusted renderer input crosses here via `settings:*` invoke channels |
| main → disk (electron-store / safeStorage) | Encrypted key ciphertext persisted under userData |
| process.env / .env → main | Dev-fallback key source (lower precedence than safeStorage, D-08) |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-06-01 | Information Disclosure | `getKeys` IPC return | mitigate | Returns presence booleans only; decrypted key never crosses IPC outbound (RESEARCH Pattern 4) |
| T-06-02 | Information Disclosure | key in logs/errors | mitigate | saveKeys/store never log the key; api-key-store has no log line referencing plaintext |
| T-06-03 | Information Disclosure | plaintext on disk | mitigate | safeStorage.encryptString -> store base64 ciphertext ONLY; isEncryptionAvailable guard |
| T-06-04 | Elevation of Privilege | settings renderer XSS → Node | mitigate | contextIsolation:true + sandbox:true on the settings window; preload throws if contextIsolation off |
