TranscriptBuffer target shape (RESEARCH Pattern 3 lines 284-326):
- `interface ITranscriptSegment { text: string; at: number; }` (at = epoch ms).
- Constructor injects a clock: `constructor(private readonly now: () => number = Date.now)`.
- `appendFinal(text: string): void` — push { text, at: now() }, clear interim, prune().
- `setInterim(text: string): void` — replace (never accumulate).
- `clear(): void` — empty finals + interim (TRN-04 hotkey target).
- `recentSince(ms: number): string` — join finals newer than now()-ms (Phase 5 span read).
- `renderable(): { finalText: string; interimText: string }` — for the overlay push (04-04 reads this).
