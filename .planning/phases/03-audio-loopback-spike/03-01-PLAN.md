Channel is single-purpose and write-only; no general renderer->main control surface added. |
| T-03-03 | Information Disclosure | captured loopback audio (renderer AudioContext) | accept | No network egress and no disk persistence in this phase (Deepgram is Phase 4); audio exists only transiently in renderer memory for RMS. contentProtection (OVL-04) keeps the overlay out of screen-share. Acceptable for a local, no-network audio spike. |
| T-03-04 | Information Disclosure | jedi:status / jedi:audio-level payloads | accept | Only a 0..1 RMS scalar crosses IPC — no audio samples, no secrets. Consistent with the established non-secret status channel. |
| T-03-SC | Tampering | npm/pip/cargo installs | accept | No new package-manager installs in this plan — built-in loopback is native to Electron (naudiodon is only pulled in on a NO-GO, deferred). No package legitimacy gate required. |
