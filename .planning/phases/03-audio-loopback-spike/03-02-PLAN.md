<objective>
Run the 03-01 app on the target Windows 11 machine at the pinned Electron version and human-judge whether system-audio loopback produces real, non-silent audio (AUD-02) — the GO/NO-GO gate that unblocks Phase 4. Output: A committed gate record with the machine, Electron version, sources tested, observed RMS behavior (playing vs paused) for each source, and a GO / partial-NO-GO / NO-GO decision — plus the same decision logged in STATE.md and PROJECT.md.
