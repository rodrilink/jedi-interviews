| packaged .exe -> OS | The portable `.exe` runs from a user-chosen location; native `.node` files load from `app.asar.unpacked/` within the app dir |
| user -> SmartScreen | An unsigned portable `.exe` triggers a Windows SmartScreen warning the user clicks through |

| Threat ID | Category | Component | Disposition | Mitigation Plan |
